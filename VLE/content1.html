Programming Basics

This page covers the basics of programming, such as variables and control flow.

What are variables?

Variables are the names you give to computer memory locations which are used to store values in a computer program.

For example, assume you want to store two values 5 and 15 in your program and at a later stage as well as a third variable 'hello', you want to use these two values.

Firstly, variables need to be declared. For the example of 5 and 15 above, this is done simply in Python:

a = 5

b = 15

x = 'hello'

This has created the variable 'a' with the value of 5, and the variable 'b' with the value of 15, as well as a variable 'x' with the string 'hello'. Numbers and strings are the main values for variables.

These variables can now be utilised as needed. For example, the value of the two variables may want to be added together, or maybe you want to take the value of 'a' away from 'b'. This can be achieved in Python by:

a + b

b - a

This has completed the intended addition and subtraction as intended.

However, what if you then want to utilised the results at a later time? You could run the addition or subtraction again... Or you could assign the result to a variable. Using the knowledge you have learnt here, what is the correct way to assign the output of the addition of the two variables to another variable?

Control Flow

Using the above information, you should be able to create simple programs that perform different functions. By intergrating control flows, you should be able to create even more complex programs.

By utilising conditional operators, three different types of control flow can be used. Firstly, if statements can be used if something fits a criteria. If more than one criteria is present, elif can be used. For any result that does not meet a criteria. Else is used. An example is shown below:

if a + b == 20

print 'Yes!'

elif a + b == 40

print 'Opps!'

else

print 'Oh no!'

Given that 'a' still equals 5 and 'b' still equals 15, the given output of the if statement above will be 'Yes!' since the value does equal 20. This is just one example of the use of if statements.

The next two control flows create loops. Loops allow for continual cycling of code until certain requirements are met, or while certain requirements are met. These are while and for loops. These work in the same form as if statement. While will run code only while the condicition is met, and for will run to a given condition. Below is an example of a while loop.

x = 5

y = 0

z = 10

while y =< x

z+=2

y+=1
